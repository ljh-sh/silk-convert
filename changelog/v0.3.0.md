# v0.3.0 (2026-06-25)

## Major

- **Repo renamed** from `ljh-sh/silk-convert` to [`ljh-sh/silk`](https://github.com/ljh-sh/silk).
  The binary was already named `silk`; now the repo matches.
  Old URL `https://github.com/ljh-sh/silk-convert` auto-redirects to the new one.

## New

- **GitHub Pages site** at https://ljh-sh.github.io/silk (install / usage / formats / FAQ)
- **GitHub Actions CI** — PR check on Ubuntu + macOS
- **GitHub Actions release** — tag-triggered cross-platform build:
  - macOS (Apple Silicon + Intel)
  - Linux (x86_64 gnu + musl)
  - Windows (x86_64)
- **CODEOWNERS** + **CONTRIBUTING.md** — PR-based workflow

## Docs

- README updated to reflect new repo name + comparison table
- SECURITY.md + COMPARISON.md reference updated

## Not in this release

- MP3/Opus/AAC/FLAC support (planned v0.4 via symphonia)
- Resampling (planned v0.4)
- `x silk` x-cmd module (planned once release pipeline stabilizes)
